**Display Driver Uninstaller (DDU)** — это утилита для глубокого удаления драйверов видеокарт (NVIDIA, AMD, Intel) и звуковых карт (Realtek, Creative) из системы. Она полностью очищает реестр, папки и файлы, оставшиеся после стандартного деинсталлятора, помогая решить проблемы с «синими экранами» (BSOD), зависаниями и падением FPS.

## ✨ Основные возможности

*   **Полная очистка:** Удаляет ключи реестра, файлы, папки и кэш драйверов, которые обычный установщик оставляет в системе.
*   **Поддержка всех популярных GPU:** Полная совместимость с драйверами **NVIDIA**, **AMD** и **Intel**.
*   **Очистка аудиодевайсов:** Поддержка удаления звуковых драйверов Realtek и Creative.
*   **Защита от обновлений Windows:** Автоматически блокирует Windows Update от скачивания старых драйверов во время процесса очистки.
*   **Логирование:** Создает подробный лог-файл каждого действия на случай непредвиденных ошибок.

## 🚀 Инструкция по правильному использованию

Для достижения наилучшего результата следуйте этому алгоритму:

1.  **Скачайте** актуальную версию DDU.
2.  **Загрузите Windows в Безопасном режиме (Safe Mode)** — это критически важно, чтобы система не блокировала удаляемые файлы.
3.  Запустите DDU.
4.  В правой панели выберите **Тип устройства** (Видеокарта) и **Устройство** (NVIDIA, AMD или Intel).
5.  Нажмите кнопку **«Удалить и перезагрузить»** (Рекомендуется).
6.  После перезагрузки установите новый чистый драйвер.
